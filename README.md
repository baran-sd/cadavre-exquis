# 🌸 Cadavre Exquis: AI Image Agent

A specialized AI agent for high-quality image generation via text-to-image prompt engineering. 

Built with 💜 for the [Pollinations.ai](https://pollinations.ai) ecosystem.

## 🚀 Features
- **BYOP (Bring Your Own Pollen)**: Keep your balance safe! Users log in with their own Pollinations account.
- **Agent Intelligence**: Integrated LLMs (Gemini, Grok, GPT-4o) to enhance and reimagine your prompts.
- **Metamorphosis Engine**: Specialized "Cadavre Exquis" mode for tri-style gradient art.
- **Chain Generation**: ⛓️ A new "Exquisite Corpse" gameplay mode where the AI continues the previous generation's story.
- **Premium UI**: Ultra-clean Glassmorphism design with responsive controls.

## 🛠 Tech Stack
- **HTML5/CSS3**: Vanilla CSS with modern Flexbox/Grid.
- **Vanilla JavaScript**: Zero dependencies for maximum performance.
- **Pollinations APIs**:
  - `/text/` for LLM prompt engineering.
  - `/p/` for Image generation (Flux, Z-Image).
  - `/account/balance` for real-time Pollen tracking.
